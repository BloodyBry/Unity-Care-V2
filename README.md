# Unity Care Clinic - Admin Panel

A web application for managing a clinic, including appointments, prescriptions, medications, and users. Built with **PHP**, **PDO**, and **MySQL**.

---

## Features

### Authentication
- Login/logout system with **role-based access** (admin, doctor, patient).
- Secure sessions.

### Admin Dashboard
- Quick links to **Appointments**, **Prescriptions**, **Medications**.
- Dashboard only accessible to admins.

### Appointments
- Create, edit, delete appointments.
- Assign patients and doctors.
- Status management: **scheduled**, **done**, **cancelled**.
- Responsive, modern table UI with hover effects.

### Medications
- Add, edit, delete medications.
- Table with IDs, names, and descriptions.
- Same UI as appointments for consistency.

### Prescriptions
- Assign medications to patients.
- Create, edit, delete prescriptions.
- Shows **doctor, patient, medication name, dosage, and date**.
- Responsive, modern table UI like appointments and medications.
